# 🎵 Mood Playlist Generator

A Spotify-style music web app that generates playlists based on user mood and language preferences using the iTunes API.

Users can select moods and languages to discover songs with album covers, artist details, and playable previews.

---

## ✨ Features

* Mood-based playlist generation
* Language selection
* Real song fetching using Deezer API
* Song previews with audio player
* Album cover display
* Attractive Spotify-style UI
* Responsive design

---

## 🌍 Supported Languages

* English
* Hindi
* Tamil
* Malayalam
* Telugu
* French

---

## 😊 Supported Moods

* Sad
* Chill

* Motivation
